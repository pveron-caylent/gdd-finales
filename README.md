# GDD Finales - Database Subject Exams

This repository contains database subject exam files with their resolutions.

## Structure

Each exam is stored in a separate file with its corresponding resolution.

## Exams

- [Exam 1](exams/exam-01.md) - First exam with resolution

## How to Use

Each exam file contains:
1. The exam questions
2. A resolution section with detailed answers

## Contributing

When adding a new exam:
1. Create a new file in the `exams/` directory
2. Follow the template format (see existing exams)
3. Include both questions and resolutions
